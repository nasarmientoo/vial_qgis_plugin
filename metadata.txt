description=Edit, validate and normalize road arc nomenclature attributes following the Colombian road naming standard (IGAC/DANE).
about=A QGIS plugin that provides a specialized workflow for standardizing the attribute table of road network layers according to the Colombian official nomenclature schema. Features include: field mapping dialog to map existing fields to 17 standard VIAL fields, an attribute editor dock with dropdown editors for controlled-vocabulary fields, per-field validation with automatic normalization, contiguous street chain detection with conflict highlighting, generator road calculator, automatic dual map labeling, and full undo/redo support via the QGIS edit buffer.
